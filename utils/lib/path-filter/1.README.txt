## 概要

solution-finderのpathの結果を基に、パーフェクトクリア成功率を維持したまま最も少ない組み合わせをみつけます。
このプログラムによって得られる組み合わせは、厳密に最小とは限らないことにご注意ください。

このプログラムは、実行するたびに結果が変わる可能性があります。
そのため、何回か実行してみて一番良いものを選ぶことを推奨します。


## セットアップ

1. solution-finderをダウンロードしてください。

2. このディレクトリにある"path-filter.jar" を、solution-finderの"output"ディレクトリ に移動させてください。



## 基本的な使い方

1. solution-finderのディレクトリでPowerShellを開きます。

* エクスプローラーでsolution-finderのsfinder.jarがあるディレクトリを開き、
 「Shiftキーを押しながら右クリック」→「PowerShellウィンドウをここで開く」。


2. solution-finderのpathコマンドを実行します。
   地形・パターン・ホールド・ハードドロップなどの設定はご自由に変更できます。

<コマンド例>
	java -jar sfinder.jar path -f csv -k pattern -t v115@zgB8EeE8EeE8EeE8FeF8CeC8JeAgH -p '*!' -c 5

<注意事項>
    * 必ず CSV(pattern) を出力するように設定してください


3．PowerShellでsolution-finderの"output"ディレクトリを開きます

<コマンド例>
	cd output

* 最後の行に "PS ***\output>" (***はsolution-finderのあるパス)が表示されればOK


4. 絞り込みを実行します。
   コマンド例の数値は設定値で、自由に変更することができますが、基本的にこの数値で問題ありません。
   詳しく知りたい方は "2.DETAILS.txt" ファイルに記載しているので、そちらをご参照ください。

<コマンド例>
	java -jar path-filter.jar 5.0 1.0 3.0 300000

<注意事項>
    * 最終結果はoutput.txtに記録されています。コマンドを再実行すると、過去の結果は失われるためご注意ください。


5. 画面上に表示される "best" の値が動かなくなったと感じたら、プログラムを停止してください。
   なお、プログラムは Ctrl+Z で停止できます。
   こちらの停止すべきタイミングについても、 "2.DETAILS.txt" ファイルに記載しています。


6. 最終結果として、組み合わせをまとめたテト譜を表示します。
   このとき、画面に "OK" と表示されていることを確認してください。
   表示されていない場合、すべてのツモをカバーできていない組み合わせなのでご注意ください。

<コマンド例>
	java -jar path-filter.jar verify
